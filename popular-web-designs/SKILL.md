# Popular Web Designs — 54 real design systems (Stripe, Linear, Vercel) as HTML/CSS

## Overview
This skill provides access to the visual languages of 54 real-world design systems. Use these templates to capture a site's color palette, typography, component styles, and shadows.

## How to Use
1.  Identify the desired aesthetic (e.g., "minimal dark mode", "vibrant landing page").
2.  Choose a matching design system from the catalog.
3.  Implement the design tokens (colors, fonts, spacing) in your HTML/CSS output.

## Font Substitution Reference
Since most sites use proprietary fonts, use these Google Fonts substitutes:
*   **Geist / Geist Sans:** Geist
*   **sohne-var (Stripe):** Source Sans 3
*   **Berkeley Mono:** JetBrains Mono
*   **Airbnb Cereal / Circular:** DM Sans
*   **UberMove:** DM Sans
